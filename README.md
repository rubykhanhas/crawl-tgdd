## Getting Started

### First, run the installer:

```bash
yarn install
# or
npm install
```

### Run app in development mode:

```bash
yarn dev
```

### Build the app:

```bash
yarn build
```

### Run app in production mode after build:

```bash
yarn start && yarn serve
```
